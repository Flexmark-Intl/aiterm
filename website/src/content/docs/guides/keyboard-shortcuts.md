## General

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+W` | Close tab (or pane if last tab) |
| `Cmd+1`â€“`9` | Switch to tab |
| `Cmd+Shift+[` | Previous tab |
| `Cmd+Shift+]` | Next tab |
| `Cmd+Shift+T` | Duplicate tab |
| `Cmd+Shift+R` | Reload tab (duplicate + close) |
| `Cmd+D` | Split pane (duplicate tab) |
| `Cmd+N` | New workspace |
| `Cmd+,` | Preferences |
| `Cmd+/` | Help |

## Editor

| Shortcut | Action |
|----------|--------|
| `Cmd+O` | Open file in editor tab |
| `Cmd+S` | Save file |
| `Cmd+F` | Find/replace |

## Notes

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+N` | Toggle notes panel |

:::note
On Windows and Linux, replace `Cmd` with `Ctrl`.
:::
